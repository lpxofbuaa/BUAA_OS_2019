# BUAA-2019操作系统课程设计源码

本仓库为2019操作系统课程设计源码，本人完成的挑战性任务为lab4-challenge多线程与信号量

实验为迭代式开发实验，所有lab分支与课上测试分支均在仓库中，可以通过checkout查看，部分课上测试代码不保证正确性。具体分支如下：

- lab0：Linux入门
- lab1：支持printf
- lab2：内存管理
- lab3：进程管理
- lab4：系统调用与fork，ipc
- lab5：文件系统
- lab6：小型shell
- lab4-challenge：挑战性任务，参照POSIX标准支持多线程与信号量机制，实现简易PV操作
- 其余分支为测试分支与课上测试分支

代码仅供参考。
